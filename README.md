# Smart Student Tracking System

Overview

This project is an IoT-based tracking system using ESP32, GPS, and GSM modules to monitor student location and provide emergency alerts.

Components

* ESP32
* GPS Module
* GSM Module
* Buzzer
* Push Button

Features

* Real-time tracking
* Emergency alert button
* SMS notification
* Buzzer alert

Working

The ESP32 reads GPS data and sends location via GSM. On pressing the button, an emergency alert is triggered.
